Welcome to SimpleReveal.

A simple ScrollReveal or AOS alternative to implement simple section or element reveal on scroll. The functionality used javascript intersectionObserver API to check weather the section or element is visible on user screen or not. Just include simrev.css and simrev.js to your html and add `simrev-up`, `simrev-down`, `simrev-right`, `simrev-left`, `simrev-up-delay`, `simrev-right-delay` etc. classes to controll how your element revealed on scroll.

Usage:

To use SimpleReveal, you need to include the simrev.css and simrev.js files in your HTML. You can then add the following classes to your elements:

  • simrev-up - Reveals the element from the bottom to the top.
  • simrev-down - Reveals the element from the top to the bottom.
  • simrev-right - Reveals the element from the left to the right.
  • simrev-left - Reveals the element from the right to the left.
  • class="simrev-up-delay" style="--delay:400ms;simrev-up-delay - similar to `simrev-up` but you can add delay before the element reveal by adding custom property 'delay' to the element you added `simrev-up-delay` (see example at the ending and see how you can use it to make interesting animation in the next section.) .
  • simrev-right-delay - similar to `simrev-right` but you can add delay before the element reveal by adding custom property 'delay' to the element you added `simrev-right-delay` .

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>SimRev Demo</title>
        <link rel="stylesheet" href="simrev.css">
    </head>
<body>
    <!-- make sure containers to get full or enough
        height so you can visualize the scroll reveal effect -->
    <div class="container">
        <h1>Hello</h1>
    </div>
    <div class="container">
        <h1>Hello 1</h1>
    </div>
    <div class="container">
        <h1>Hello 2</h1>
    </div>
    <script src="simrev.js"></script>
</body>
</html>
